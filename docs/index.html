Enodios

Ἐνόδιος — Hermes of the road. One-script Linux setup to run Hermes Agent on your NVIDIA GPU with vLLM. Fast tool calling. No cloud inference keys.

GitHub Quick start Requirements

How it works

You │ ▼ Hermes Agent ─── tools: terminal, files, web, memory │ │ OpenAI API http://127.0.0.1:8000/v1 ▼ vLLM (enodios) ─── Hermes 3 AWQ + tool-call parser │ ▼ NVIDIA GPU

Enodios installs vLLM natively (no Docker), starts an OpenAI-compatible server, and configures Hermes Agent to use it. Default model: uncensored hermes3:8b with reliable tool calls.

Quick start

  1. Install Hermes Agent (if needed) 
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
hermes setup
  2. Install Enodios + vLLM 
    curl -fsSL https://raw.githubusercontent.com/DataKnifeAI/enodios/main/install.sh | bash
enodios install
  3. Check GPU stack 
    enodios doctor
  4. Start inference (keep terminal open) 
    enodios start
  5. Wire Hermes + chat 
    enodios configure
hermes chat

Troubleshooting

VRAM full? Close GPU apps or export ENODIOS_GPU_UTIL=0.55 before enodios start.

Port 8000 busy? export ENODIOS_PORT=8001 and re-run configure.

No nvcc? Fine — enodios uses PyTorch sampler fallback. Install cuda toolkit only for optional speedup.
